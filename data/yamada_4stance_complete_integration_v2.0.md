## 🎯 4スタンス理論とは?

### 定義
人間の体の使い方を**2軸×2軸=4タイプ**に分類する理論

### 2つの判定軸
1. **縦軸: アッパー型 vs ダウン型**
   - 重心位置(つま先 vs かかと)
   - 蹴り足(前足 vs 後ろ足)
   - 固定軸と動作軸の違い

2. **横軸: クロス型 vs ストレート型**
   - 体の使い方(捻る vs 回す)
   - 上半身と下半身の関係(分離 vs 一体)

## 🔬 軸理論(5つの軸ポイント)

### 5つの軸ポイント
1. **足** (foot)
2. **膝** (knee)
3. **骨盤** (pelvis)
4. **みぞおち** (solar plexus / diaphragm)
5. **肩** (shoulder)

### タイプ別の軸の使い方

#### アッパー型(A1, A2)
- **固定軸**: 膝 + みぞおち
- **動作軸**: 骨盤 + 肩
- **意味**: 膝とみぞおちを固定して、骨盤と肩を動かして力を生む

#### ダウン型(B1, B2)
- **固定軸**: 骨盤 + 肩
- **動作軸**: 膝 + みぞおち
- **意味**: 骨盤と肩を固定して、膝とみぞおちを動かして力を生む

## 🎯 タイプ別詳細特性

### A1型: アッパー・クロス型(ジョコビッチ)

#### 基本身体特性
| 項目 | 特徴 |
|------|------|
| 重心 | つま先重心 = 前重心 |
| 蹴り足 | 前足から蹴り出す |
| 固定軸 | 膝 + みぞおち |
| 動作軸 | 骨盤 + 肩 |
| パワー生成 | 筋肉を伸ばす(伸張反射) |
| 体の使い方 | 捻る(上半身↔下半身逆方向) |
| 打球軌道 | 上向き45度 |
| トス位置 | 前方(体より前) |
| 典型的ミス | ネット(前に込みすぎ) |

#### サーブ構造(8段階)
1. **構え**: やや前傾、下半身と上半身が逆向き(捻り構え)
2. **トス**: 前方トス、骨盤+肩が動く、手首を効かせる
3. **テークバック**: 前足重心、肩と腰が逆方向(捻り)
4. **トロフィーポーズ**: かなり捻れている、前足に重心
5. **蹴り出し**: 前足から蹴る
6. **インパクト**: ジャンプ頂点、上向き45度、右肩→左膝
7. **フォロースルー**: 上から前に、左足内側入り
8. **フィニッシュ**: 体が捻れた状態、前に倒れ込む

#### 改善ポイント
- 前重心過多に注意
- 右肩と左膝のイメージを明確に
- 捻り動作で腰固定、肩を先に回す

## 🎯 全タイプ共通原則

### 軸の原則
1. トスを上げる時、どっちかに重心を乗せる
2. テークバック〜フォロースルーまで片側の軸を維持
3. 投げたい場所を見る
4. トスを上げた後も手を残す
5. 5つの柱を壁にする

### ターゲティングの原則
1. 地面ではなく、ネットの上を狙う
2. フラット/スライス: ネット上ギリギリ(センターベルト)
3. キック/スピン: より高い弧
4. 迷ったらセンターベルトの上

### リズムの原則
1. 「ワン・ツー・スリー」でリズムを把握
2. そのリズムに合わせてトスの高さを調整
3. トスとリズムがそろったらフルスイング
4. 3球目でターゲットに入れる練習

### イメージの原則
1. サーブはボール投げ(横投げ/斜め投げ)
2. フリスビーで横投げ感覚をつかむ
3. 万歳の自然なライン(斜め上)
4. ネットの上に輪を置いたイメージ
