## 1. トスの基本原則（全タイプ共通）

### 1.1 正しいトスの高さ

**山田式核心**: "サーブってスイングの中にトスを入れるが正解"

**具体的方法**:
1. 素振りで最も振りやすいリズムを"ワン・ツー"のカウントで測る
2. トスを上げたボールが"ワン・ツー"のタイミングで打点に落ちてくる高さに調整
3. 多くの人は普段のトスが高すぎる→"ワン・ツー"リズムに合う中程度の高さが正解

**イメージ**: 電車のアナウンス「まもなく、○○駅です」→「まもなく」=トス、「○○駅」=打点

---

### 1.2 トスを安定させるコツ

**重心の片側固定法**:
- トスを上げる直前に体重を**必ず左右どちらかの足に完全に乗せる**
- 重心が中立のままではなく、片側にしっかりかけた状態で"キュッ"と投げ上げる
- 片側に体重をかけることで重心がぶれず、投げ上げる軌道が安定

**NG例**: 
- 重心がどっちつかずの状態で「ポイッ」と投げる→安定しない

**OK例**:
- 右足or左足どちらかにしっかり乗せて→「キュッ」と手首を使って上げる→トス安定

## 2. アッパータイプ vs ダウンタイプ（4スタンス理論A vs B）

### 2.1 基本的な体の使い方の違い

#### アッパータイプ（A1/A2）の特徴:
- **骨盤・肩から力を出す**（体の上部・中心から）
- **腹筋群で動作を主導**し、身体前面全体を意識
- **手首が強い**→インパクト時に手首を背屈（山折り）させて威力を出す
- **脇を閉じた状態で構える**→上半身をコンパクトに保つ
- **トスは肩や体幹を使って、腕全体で上げる**

#### ダウンタイプ（B1/B2）の特徴:
- **膝・みぞおちから力を出す**（体の下部・中心から）
- **膝を可動ポイントとして使う**→膝を前に出して体重移動
- **肘が強い**→インパクト時に肘を伸ばして威力を出す
- **脇は開いたまま**→肘をやや落としてラケットを担ぐ
- **トスは肘や手首を使って、コンパクトに上げる**

---

### 2.2 サーブのリズム（テンポ）の違い

**4タイプのリズム順序**（速い→遅い）:
1. **A2タイプ**: 最速（2拍）→ "タンメン"イメージ
2. **A1タイプ**: 中速（3拍）→ "チャーシュー麺"イメージ
3. **B1タイプ**: 中速（3拍）→ "担々麺"イメージ
4. **B2タイプ**: 最遅（4拍）→ "ラーメン"イメージ

**ポイント**: "ため"を持つか持たないかの生まれつきの違い。いい悪いではなく、個性。

---

### 2.3 サーブフォームの違い

#### A1タイプのサーブ特徴:
- 腹筋群でサーブ動作を主導し、身体前面全体を意識
- **左軸を形成**→みぞおちと左膝の接触位置で軸を作る
- 両肩を固め、肩関節のみ柔軟に保って**左側へ身体を伸展**
- **クロススタンス**で、足を先に揃えてから身体を軸に合わせる
- 右足に体重を乗せた後、左膝で踏みつけるように切り返し、左軸を強調
- みぞおちの延長線上でインパクト

#### A2タイプのサーブ特徴:
- **リズムが最速**（2拍）
- **必ず両足を揃える**（マスト）→足を揃えることでリズム安定、回転アシスト
- 脇を閉じて構え、手首背屈でインパクト

#### B1タイプのサーブ特徴:
- **膝を可動ポイント**として前に出す
- 肘を落としてラケットを担ぎ、脇は開いたまま
- 手首背屈でインパクト直前に加速
- **足の配置は自由**（揃える・揃えない両方OK）

#### B2タイプのサーブ特徴:
- **リズムが最遅**（4拍）→ "ため"が大きい
- **両足を揃えず、開いたスタンスを保つ**→広いベースで軸のブレを許容
- 膝を前に出して体重移動、肘を使って威力を出す

---

## 3. クロスタイプ vs ストレートタイプ（4スタンス理論の捻り vs 回り）

### 3.1 基本的な体の使い方の違い

#### クロスタイプ（斜め回転型）の特徴:
- **自分の軸に対して斜めに回転**する動きが得意
- **肩と腰を反対方向に捻る**（肩を固定して腰を引く）
- **サーブで脚を揃えない**（例: ジョコビッチ、ディミトロフ）
- トスは**体を捻りながら上げる**

#### ストレートタイプ（左右対称型）の特徴:
- **体を左右対称に水平的に使う**のが得意
- **肩と腰を同じ方向に回す**（肩と膝が密着して回る）
- **サーブで脚を揃える**（例: 錦織圭）
- トスは**体を回しながら上げる**

---

### 3.2 サーブのスタンス（足の配置）

| タイプ | 足の配置 | 理由 |
|--------|----------|------|
| **A2タイプ** | **必ず揃える** | リズム安定、回転アシストのため |
| **B2タイプ** | **必ず開く** | 広いベースで軸のブレを許容、揺れ幅で威力を出す |
| **A1/B1タイプ** | **自由** | 個人の感覚や目的に合わせて選択 |
| **クロスタイプ** | **揃えない** | 斜め回転を活かすため |
| **ストレートタイプ** | **揃える** | 左右対称の動きを活かすため |

## 4. スライスサーブの打ち方（全タイプ共通）

### 4.1 6ステップメソッド

**ステップ1: フリスビー投げイメージ**
- 右手でフリスビーをオーバースロー投げすると自然に**アウトサイドイン**のスイング軌道になる
- この振りがスライスサーブの基本

**ステップ2: コンチネンタル・グリップ**
- フォアハンドのボール付きと同様の握り方（コンチネンタル）でラケットを握る
- この握りが分かれば「めちゃめちゃ簡単」

**ステップ3: トスの位置を素振りで確認**
- 素振りでスライスの当て方をイメージ
- そのスイングが最も楽にできる位置にトスを上げる

**ステップ4: 体のローテーションを使う**
- 手首やコート先だけで打たず、フリスビーを投げるような振りと**身体の回転を連動**
- 「コート先でいかないように」

**ステップ5: 狙うポイントを先にイメージしてセットアップ**
- センター、ワイド等、目的地を頭で描いてから構える
- **足の向きと身体の向きを固定**→打ちたい場所にセットアップ

**ステップ6: ミスしたらイメージ修正→再セット→連続打ち**
- ネットミスや左右ずれが出たら、頭の中の打点イメージの高さ・左右を調整
- すぐにリセットして同じ動作を繰り返す

## 7. タイプ別診断ポイント（サーブ関連）

### 7.1 アッパー vs ダウンの診断

| 診断項目 | アッパー | ダウン |
|----------|----------|--------|
| **トスの上げ方** | 肩や体幹を使って、腕全体で | 肘や手首を使って、コンパクトに |
| **体重の位置** | 前寄り（つま先側） | 後ろ寄り（かかと側） |
| **力の発揮** | 骨盤・肩（体の上部） | 膝・肘（体の下部） |
| **構えの姿勢** | 脇を閉じた状態 | 脇は開いたまま |
| **サーブのリズム** | 速い（A2=2拍） | 遅い（B2=4拍） |
| **インパクト意識** | 手首背屈（山折り） | 肘伸ばし |

---

### 7.2 クロス vs ストレートの診断

| 診断項目 | クロス | ストレート |
|----------|--------|------------|
| **体の使い方** | 肩と腰を反対方向に捻る | 肩と腰を同じ方向に回す |
| **足の配置** | 揃えない | 揃える |
| **トスの上げ方** | 体を捻りながら上げる | 体を回しながら上げる |
| **動きの方向** | 斜め回転 | 左右対称（水平） |
| **サーブの軸** | 軸が捻れている | 軸が回っている |

## 8. 山田式核心表現集（サーブ関連）

1. **"サーブってスイングの中にトスを入れるが正解"**
   - トスとスイングを別々に考えず、一連の流れとして捉える

2. **"ワン・ツー"リズム**
   - 素振りのリズムに合わせてトスの高さを決める

3. **"重心がどっちつかずの状態で投げると安定しない"**
   - トス時は必ず片足に体重を完全に乗せる

4. **"フリスビーを投げるように"**
   - スライスサーブの基本イメージ

5. **"手首背屈でインパクト"**（アッパータイプ）
   - 手首を上に反らせて（山折り）打つ

6. **"肘を伸ばしてインパクト"**（ダウンタイプ）
   - 肘の伸展で威力を出す

7. **"肩と腰を反対方向に捻る"**（クロスタイプ）
   - 体の捻転差で力を生む

8. **"肩と腰を同じ方向に回す"**（ストレートタイプ）
   - 体全体の回転で力を生む

## 9. タイプ別サーブ推奨事項まとめ

### A1タイプ（アッパー・クロス）:
- **トス**: 肩から腕全体で上げる、体を捻りながら
- **スタンス**: クロススタンス、足の配置は自由
- **リズム**: 中速（3拍）
- **体の使い方**: 左軸を形成、左側へ伸展、肩と腰を反対方向に捻る
- **インパクト**: 手首背屈、みぞおちの延長線上

### A2タイプ（アッパー・ストレート）:
- **トス**: 肩から腕全体で上げる、体を回しながら
- **スタンス**: 必ず両足を揃える（マスト）
- **リズム**: 最速（2拍）
- **体の使い方**: 脇を閉じて構え、肩と腰を同じ方向に回す
- **インパクト**: 手首背屈

### B1タイプ（ダウン・クロス）:
- **トス**: 肘や手首でコンパクトに上げる、体を捻りながら
- **スタンス**: 足の配置は自由
- **リズム**: 中速（3拍）
- **体の使い方**: 膝を前に出す、肘を落としてラケットを担ぐ、脇は開く、肩と腰を反対方向に捻る
- **インパクト**: 肘伸ばし、手首背屈で加速

### B2タイプ（ダウン・ストレート）:
- **トス**: 肘や手首でコンパクトに上げる、体を回しながら
- **スタンス**: 必ず両足を開く
- **リズム**: 最遅（4拍）→ "ため"が大きい
- **体の使い方**: 膝を前に出す、広いベースで軸のブレを許容、肩と腰を同じ方向に回す
- **インパクト**: 肘伸ばし

## 10. サーブ診断質問（AIシステム用）

### 質問1: トスの上げ方
**質問**: サーブのトスを上げるとき、どのように上げますか？
**選択肢**:
1. 肩や体幹を使って、腕全体で上げる → アッパー傾向
2. 肘や手首を使って、コンパクトに上げる → ダウン傾向
3. 特に意識していない
4. わからない

---

### 質問2: サーブのリズム
**質問**: サーブを打つとき、"ため"をどのくらい作りますか？
**選択肢**:
1. ほとんど"ため"を作らず、素早く打つ（2拍） → A2傾向
2. 少し"ため"を作って打つ（3拍） → A1/B1傾向
3. しっかり"ため"を作って打つ（4拍） → B2傾向
4. 意識したことがない

---

### 質問3: サーブのスタンス
**質問**: サーブを打つとき、足をどのように配置しますか？
**選択肢**:
1. 必ず両足を揃える → A2/ストレート傾向
2. 必ず両足を開く → B2傾向
3. 状況に応じて変える → A1/B1傾向
4. 意識したことがない

---

### 質問4: 体の使い方（捻り vs 回り）
**質問**: サーブを打つとき、体をどのように使いますか？
**選択肢**:
1. 肩と腰を反対方向に捻って打つ → クロス傾向
2. 肩と腰を同じ方向に回して打つ → ストレート傾向
3. 特に意識していない
4. わからない

---

### 質問5: インパクトの意識
**質問**: サーブを打つとき、インパクトでどこを意識しますか？
**選択肢**:
1. 手首を反らせて（背屈させて）打つ → アッパー傾向
2. 肘を伸ばして打つ → ダウン傾向
3. 特に意識していない
4. わからない
