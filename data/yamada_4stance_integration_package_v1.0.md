## 🎯 4スタンス理論・完全マトリクス

### **縦軸: アッパー vs ダウン**
```
アッパー型の特徴:
├─ 重心: つま先/前重心
├─ 固定軸: 膝 + みぞおち
├─ 動作軸: 骨盤 + 肩
├─ 蹴り足: 前足から蹴る
├─ パワー: 筋肉を伸ばす(伸張反射)
├─ 打球: 上向き45度
└─ 典型的ミス: ネット

ダウン型の特徴:
├─ 重心: かかと/後ろ重心
├─ 固定軸: 骨盤 + 肩
├─ 動作軸: 膝 + みぞおち
├─ 蹴り足: 後ろ足から蹴る
├─ パワー: 筋肉を縮める(収縮力)
├─ 打球: 下向き打ち下ろし
└─ 典型的ミス: オーバー
```

### **横軸: クロス vs ストレート**
```
クロス型の特徴:
├─ 体の使い方: 捻る(上半身↔下半身逆方向)
├─ イメージ: 右肩→左膝がひっつく
└─ フィニッシュ: 左足が内側に入り、体が捻れる

ストレート型の特徴:
├─ 体の使い方: 回す(上半身=下半身一体)
├─ イメージ: 肩と膝が一緒に動く
└─ フィニッシュ: 体が正面を向く
```

### **4タイプ分類**

|  | **クロス型** | **ストレート型** |
|---|---|---|
| **アッパー型** | **A1: ジョコビッチ**<br>前足蹴り・捻って伸ばす | **A2: ナダル・錦織**<br>前足蹴り・回して伸ばす |
| **ダウン型** | **B1: フェデラー・山田**<br>後ろ足蹴り・捻って縮める | **B2: デルポトロ・バブリンカ**<br>後ろ足蹴り・回して縮める |

### **タイプ判定ロジック**

```python
def determine_type(part1_answers, part2_answers):
    """
    4スタンスタイプ判定
    
    Args:
        part1_answers: パート1の回答リスト ["A", "B", "A", ...]
        part2_answers: パート2の回答リスト ["A", "B", "A", ...]
    
    Returns:
        type_code: "A1", "A2", "B1", "B2"
        type_name: "アッパー・クロス", etc.
        pro_example: "ジョコビッチ", etc.
    """
    # パート1: アッパー vs ダウン判定
    upper_count = part1_answers.count("A")
    down_count = part1_answers.count("B")
    
    if upper_count > down_count:
        vertical_type = "upper"
        vertical_name = "アッパー型"
    else:
        vertical_type = "down"
        vertical_name = "ダウン型"
    
    # パート2: クロス vs ストレート判定
    cross_count = part2_answers.count("A")
    straight_count = part2_answers.count("B")
    
    if cross_count > straight_count:
        horizontal_type = "cross"
        horizontal_name = "クロス型"
    else:
        horizontal_type = "straight"
        horizontal_name = "ストレート型"
    
    # タイプコード決定
    type_matrix = {
        ("upper", "cross"): {
            "code": "A1",
            "name": "アッパー・クロス型",
            "pro": "ジョコビッチ"
        },
        ("upper", "straight"): {
            "code": "A2",
            "name": "アッパー・ストレート型",
            "pro": "ナダル、錦織圭"
        },
        ("down", "cross"): {
            "code": "B1",
            "name": "ダウン・クロス型",
            "pro": "フェデラー、山田秀"
        },
        ("down", "straight"): {
            "code": "B2",
            "name": "ダウン・ストレート型",
            "pro": "デルポトロ、バブリンカ"
        }
    }
    
    return type_matrix[(vertical_type, horizontal_type)]
```
