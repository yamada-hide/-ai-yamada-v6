## 📋 **使い方**

### **推定方法:**
```
会話の中で、パート1から2-3問、パート2から2-3問を混ぜる
↓
ユーザーの回答から、A型/B型、クロス/パラレルを推定
↓
組み合わせて判定:
- A型 + クロス = A-1
- A型 + パラレル = A-2
- B型 + クロス = B-1
- B型 + パラレル = B-2
```

### **推定精度:**
- **パート1（Upper/Down）**: 44問利用可能
  - 2-3問 → 推定精度 75-80%
  - 5-6問 → 推定精度90-95%
- **パート2（Cross/Parallel）**: 38問利用可能

  - 5-6問 → 推定精度90-95%
- **総合精度**: 各パート5-6問、計10-12問 → 推定精度95%以上

### **重要な前提**
- **アッパー（A型）**: **前が前提**（前足重心、打点も前）
- **ダウン（B型）**: **後ろが前提**（後ろ足重心、打点も後ろ）
