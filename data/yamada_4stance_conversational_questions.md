# 山田式4スタンス「さりげなく推定」質問集 v1.0

**作成日**: 2025-12-09  
**目的**: 会話の中でさりげなく4スタンスタイプを推定するための質問集

---

## 📋 **使い方**

### **推定方法:**
```
会話の中で、パート1から2-3問、パート2から2-3問を混ぜる
↓
ユーザーの回答から、A型/B型、クロス/パラレルを推定
↓
組み合わせて判定:
- A型 + クロス = A-1
- A型 + パラレル = A-2
- B型 + クロス = B-1
- B型 + パラレル = B-2
```

### **推定精度:**
- **パート1（Upper/Down）**: 30問利用可能
  - 2-3問 → 推定精度 75-80%
  - 5-6問 → 推定精度85-90%
- **パート2（Cross/Parallel）**: 28問利用可能
  - 2-3問 → 推定精度 75-80%
  - 5-6問 → 推定精度85-90%
- **特に重要**: 質問41（肩と膝のひっつき方）、質問42（バランスの取り方）、質問45（可動域の感覚）は単独で高精度判定が可能

**解答の選択肢:**
- A. 左肩と右膝がひっつきに行く（対角線上） → **クロス型**
- B. 右肩と右膝がひっつく（同じ側） → **パラレル型（ストレート）**

### **1. 質問の混ぜ方**

- 「ちなみに、〇〇の時ってどうしてる？」のような自然な聞き方

### **2. 推定精度の向上**
- 質問数が少ない場合（2-3問）: 推定精度70-75%
- 質問数が多い場合（5-6問）: 推定精度80-85%
- 確信度が低い場合: 「もう少し教えてもらってもいい？」と追加質問

- 「診断してます」と悟られないように
- 普通の対話の中で、さりげなく情報収集
- 推定結果は内部的に保持
