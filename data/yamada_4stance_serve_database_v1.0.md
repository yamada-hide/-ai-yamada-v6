## 🎯 タイプ別詳細データ

### **A1型: アッパー・クロス型(ジョコビッチ)**

#### 基本身体特性
- **重心タイプ**: つま先重心 = 前重心
- **固定軸**: 膝 + みぞおち
- **動作軸**: 骨盤 + 肩
- **パワー生成**: 筋肉を伸ばす(伸張反射)
- **体の使い方**: 捻る(上半身↔下半身逆方向)

#### サーブ構造
**【構え〜トス】**
- 構え姿勢: やや前傾、つま先に重心
- 上下半身の向き: 下半身→正面/横、上半身→逆方向(捻り構え)
- トス位置: 前方(体より前)
- トス時重心: 前足側
- トス手の動き: 骨盤+肩が動く、手首を効かせる
- トス後の手: 前方に伸びる

**【テークバック〜トロフィー】**
- テークバック重心: 前足側
- 肩と腰の関係: 逆方向に動く(捻り分離)
- トロフィーポーズ: かなり捻れている
- 蹴り足準備: 前足に重心が乗る

**【インパクト】**
- 蹴り足: 前足から蹴り出す
- ジャンプタイミング: 頂点または上昇中
- 打球軌道イメージ: 上向き45度
- スイング加速: インパクト前(上昇中)
- 右肩-左膝関係: 右肩→左膝がひっつくイメージ
- 手首使用: 手首を効かせる
- インパクト位置: 前方の高い打点

**【フォロースルー】**
- 振り抜き方向: 上から前に
- 体の回転: 上半身が先、下半身が追う(分離回転)
- 左足の動き: 内側(右側)に入る
- フィニッシュ姿勢: 体が捻れた状態
- 最終重心: 前に倒れ込む
- フォロースルー軸: 左足(前足)軸

#### 典型的ミスと原因
- **主なミス**: ネット
- **原因**: 前重心が過多、前に込みすぎ
- **対策**: トスを上げすぎない、後ろ足にも重心を残す意識

#### 改善ポイント
1. 右肩と左膝のイメージを明確に持つ
2. 捻り動作で腰を固定し、肩を先に回す
3. 前重心が過多にならないよう調整
4. トス位置を前過ぎない範囲で最適化
5. 上向き45度の打球イメージを強化

#### 日常動作での確認
- 立ち座り: 前傾姿勢禁止がしんどい
- ペットボトル: 手首で傾ける
- 押されるテスト: 前に押されてもバランスOK
- ハイタッチ: 上・前に動く
- スプリットステップ: ジャンプする
- ランニング: 膝を上げる、肩と膝が逆方向

#### プロ選手参考
- **ノバク・ジョコビッチ**: 完璧なA1型サーブ

**【フォロースルー】**

5. 上向き45度の打球イメージを強化

### **B1型: ダウン・クロス型(フェデラー/山田)**

#### 基本身体特性
- **重心タイプ**: かかと重心 = 後ろ重心
- **固定軸**: 骨盤 + 肩
- **動作軸**: 膝 + みぞおち
- **パワー生成**: 筋肉を縮める(収縮力)
- **体の使い方**: 捻る(上半身↔下半身逆方向)

#### サーブ構造
**【構え〜トス】**
- 構え姿勢: やや後傾、かかとに重心
- 上下半身の向き: 下半身→正面/横、上半身→逆方向(捻り構え)
- トス位置: 真上に近い(前過ぎない)
- トス時重心: 後ろ足側
- トス手の動き: 膝+みぞおちが動く、肘で受け取る
- トス後の手: 体の近くに残る

**【テークバック〜トロフィー】**
- テークバック重心: 後ろ足側
- 肩と腰の関係: 逆方向に動く(捻り分離)
- トロフィーポーズ: かなり捻れている
- 蹴り足準備: 後ろ足に重心が残る

**【インパクト】**
- 蹴り足: 後ろ足から蹴り出す
- ジャンプタイミング: 下降中
- 打球軌道イメージ: 下向き打ち下ろし
- スイング加速: インパクト後(下降中)
- 右肩-左膝関係: 右肩→左膝がひっつくイメージ
- 手首使用: 手首は固定、肘で打つ
- インパクト位置: 体の近くの打点

**【フォロースルー】**
- 振り抜き方向: 上から下に
- 体の回転: 上半身が先、下半身が追う(分離回転)
- 左足の動き: 内側(右側)に入る
- フィニッシュ姿勢: 体が捻れた状態
- 最終重心: 後ろ足で立っている
- フォロースルー軸: 右足(後ろ足)軸

#### 典型的ミスと原因
- **主なミス**: オーバー
- **原因**: 後ろ重心が過多、後ろに反りすぎ
- **対策**: トスを前過ぎない位置に、打ち下ろすイメージ強化

#### 改善ポイント
1. 右肩と左膝のイメージを明確に持つ
2. 捻り動作で腰を固定し、肩を先に回す
3. 後ろ重心が過多にならないよう調整
4. トスを真上に近い位置で安定させる
5. 打ち下ろすイメージを強化

#### 日常動作での確認
- 立ち座り: 前傾姿勢禁止が問題ない
- ペットボトル: 肘で受け取る
- 押されるテスト: 後ろに押されてもバランスOK
- ハイタッチ: 下・後ろに動く
- スプリットステップ: 沈む
- ランニング: 地面を蹴る、肩と膝が逆方向

#### プロ選手参考

## 🎯 全タイプ共通原則(動画データより)

### 軸の原則(mhnHy-zVdXo, BvFpzyp0rQoより)
1. **トスを上げる時、どっちかに重心を乗せる**
   - 両足均等NG、必ず右足 or 左足に乗せる
   - カゴにボールを入れる練習で感覚をつかむ

2. **テークバック〜フォロースルーまで片側の軸を維持**
   - テークバック: 片側に軸
   - インパクト: 左足一直線の軸
   - フォロースルー: 左足軸で完結

3. **投げたい場所を見る**
   - トスを上げる前に必ずターゲットを見る

4. **トスを上げた後も手を残す**
   - 左手を残すことで安定性向上

5. **5つの柱を壁にする**
   - 両足、両腰、頭の5つのポイントを一直線に
   - 片側に壁を作るイメージ

### ターゲティングの原則(t9VZPLKQzO0より)
1. **地面ではなく、ネットの上を狙う**
   - サーブは直線的、地面狙いは不安定

2. **フラット/スライス: ネット上ギリギリ**
   - センターベルトの上を通す
   - これより低いとポスト

3. **キック/スピン: より高い弧**
   - ネットの上の空間を広く使う

4. **迷ったらセンターベルトの上**
   - 左右のブレがあっても入りやすい
   - 目印があるので狙いやすい

### リズムの原則(4XPhn4CbJB8より)
1. **「ワン・ツー・スリー」でリズムを把握**
   - ボールを投げながら声に出す
   - 自分の自然なリズムを言語化

2. **そのリズムに合わせてトスの高さを調整**
   - 投げるリズムとトスのリズムを一致させる

3. **トスとリズムがそろったらフルスイング**
   - まず"ホームラン"で高く打つ
   - 徐々にネットの上を狙う

4. **3球目でターゲットに入れる練習**
   - 同じリズム、同じフルスイングで調整

### イメージの原則(jfJQhIcgOYM, 4XPhn4CbJB8より)
1. **サーブはボール投げ(横投げ/斜め投げ)**
   - 真上投げではない
   - 自然な体のラインは斜め

2. **フリスビーで横投げ感覚をつかむ**
   - 空中で横に切るように投げる
   - サーブの横投げ感覚と一致

3. **万歳の自然なライン**
   - 真っ直ぐ上ではなく、斜めに上がる
   - それが人間の自然な動き

4. **ネットの上に輪を置いたイメージ**
   - そこを通すイメージで直線的に打つ

## 📋 タイプ別推奨トレーニング

### アッパー型(A1, A2)共通
1. **前重心コントロール練習**
   - 後ろ足に少し重心を残す意識
   - ネットミス防止

2. **ジャンプ頂点打ち練習**
   - 小さくジャンプして頂点で打つ
   - 上向き45度のイメージ強化

3. **前足蹴り出し強化**
   - 前足で地面を蹴る感覚を磨く
   - 地面反発の利用

### ダウン型(B1, B2)共通
1. **後ろ重心コントロール練習**
   - 打ち下ろしイメージ強化
   - オーバーミス防止

2. **ジャンプ下降打ち練習**
   - 小さくジャンプして下降中に打つ
   - 下向き打ち下ろしのイメージ

3. **後ろ足蹴り出し強化**
   - 後ろ足から前足に流す感覚
   - サイドステップとの連動

### クロス型(A1, B1)共通
1. **右肩-左膝イメージ練習**
   - 立った状態で右肩と左膝を引っ付ける
   - バランス確認

2. **捻り動作強化**
   - 腰を固定して肩だけ回す練習
   - 上半身と下半身の分離

3. **左足内側入りフィニッシュ**
   - フィニッシュで左足が内側に入る感覚

### ストレート型(A2, B2)共通
1. **一体回転練習**
   - 腰と肩を一緒に回す
   - 正面から回す感覚

2. **肩と膝の同期**
   - 肩と膝が一緒に動く意識
   - ランニングで確認

3. **正面フィニッシュ**
   - フィニッシュで体が正面を向く感覚

> 「ダウンタイプの方は後ろ橋から恵安ようなイメージで打っていただいたら結構バランス良くなるかなって感じするんですよね。前足の方に重心を乗せるイメージか合うんですけどそれをやる日開く日からだ入りづらい。」(10:00-10:19)

> 「僕はダウンだから後ろ足から蹴る。アッパーは地面の反発を感じてバネで上がる感じ。」(2:05-2:37)

### 3qDiktlxlKo(クロス vs ストレート)
> 「クロスタイプというのは身体をねねじってローサを起こす方がやりやすい方。ストレートタイプっていうのは逆に体をめ子らずに上半身とか話じゃねーじゃーずにそのまま一緒に回してしまった方が力が出やすい方。」(1:48-2:04)

> 「右肩と左膝を引っ付けるイメージで。次こういうふうに立てるするとたぶんバランスが取りやすいの2分こっちだと思うか。これね体がネジれる人の特徴。」(3:05-3:29)

### A4YKspqSzCc(軸とタイプ)
> 「足と膝とみぞおちを軸上のラインに残した人は骨盤と方が粥状から外れた形になるのでこうやってつま先に重心がかかりやすいタイプの方になられます。こういう方は比較的まあサーブを打ちに行く時も前に前にご重心がかかってボールを放っキングされる方が多いかなー。」(3:13-3:37)

### BvFpzyp0rQo(軸の作り方)
> 「テークバックを入れた際もちゃんと片側に軸が乗っかっている状態、そしてフォロースルーに入った時もちゃんと片側に軸が乗っかっている状態っていうのを意識していただければな。」(transcript verbatim)

### 4XPhn4CbJB8(リズムとフォーム)
> 「まずボールを投げ好きなように…『ワンツースリー』って言って投げながら自分のリズムを数える。その後自分の投げたリズムにトスが落ちてくるように練習。」(4:35-5:04)
